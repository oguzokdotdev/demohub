# 💼 DemoHub

Интерактивное портфолио, позволяющее запускать и тестирвоать демо-проекты прямо в Telegram.

---

## 🤔 Что такое DemoHub?

**DemoHub** -- это платформа для демонстрации моих разработок.
Посетитель сможет по нажатию пары кнопок запустить демо-бота на ограниченное время, чтобы посмотреть, как он работает.

*Боты будут автоматически выключаться по истечению 10-ти минутного таймера, чтобы напрасно не тратить ресурсы сервера.*

---

## 💡 Идея проекта

**Идея проекта** -- дать потенциальным закзачикам возможность **"потрогать"** ботов своими руками, а не просто прочитать описание и посмотреть пару скучных скриншотов.

Это делает портфолио более **наглядным и интересным.**

---

## 🚀 Развитие и запуск проекта

Запуск проекта планируется в 3 этапа:

*1. Сайт-портфолио (сейчас)*

Небольшая веб-страница с кратким описанием моей деятельности и парой кейсов.
Доступно по этой ссылке:

> *🌐 https://www.oguzok.tech/*

Если с DNS или доменом что-то случится, эта страница всегда будет доступна по [этому](https://demohub.pages.dev/) адресу.

2. DemoHub MVP

Telegram-бот с основным функционалом проекта:

- Главная страница (приветствие)
- Список демо-проектов (портфолио)
- Запуск демо-проектов с автоматическим выключением
Собственный API для управления демо-проектами.

3. DemoHub Release

В релизной версии проекта будут осуществлены:

- Основной Telegram-бот
- Панель управления (TMA) с авторизацией, ролями, управлением проектами (добавление, удаление, измененение)
Без авторизации будет доступно только включение демо.
Остальной функционал (администрирование) будет доступен в режиме просмотра.
